## docker run

| 选项 | 含义                                                         | 例子                       |
| ---- | ------------------------------------------------------------ | -------------------------- |
| -i   | 可以给容器中的进程附加一个标注输入                           | docker run -i debian ls    |
| -t   | 可以给容器当中的进程分配一个伪终端，这会让进程的输入输出功能更为完整 | docker run -i -t debian ls |
|      |                                                              |                            |
