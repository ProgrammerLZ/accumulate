| 参数                            | 作用                                                         |
| ------------------------------- | ------------------------------------------------------------ |
| -Xms                            | 最小堆内存                                                   |
| -Xmx                            | 最大堆内存                                                   |
| -XX:+HeapDumpOnOutOfMemoryError | 在出现内存溢出异常时Dump出当前的内存堆转储快照以便事后进行分析 |
| -Xoss                           | 设定本地方法栈的大小，注意：对于Hotspot来说，这个参数没有作用 |
| -Xss                            | 设定栈内存容量                                               |
| -XX:MaxPermSize                 | 方法区最大容量                                               |
| -XX:PermSize                    | 方法区最小容量                                               |
| -XX:MaxDirectMemorySize         | 直接内存最大容量                                             |
|                                 |                                                              |
