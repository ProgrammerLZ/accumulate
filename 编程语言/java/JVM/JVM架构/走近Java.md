## Java技术体系

![](走近Java/2167990.jpg)

Java如果**按照功能**划分，可以这样划分：

* Java程序设计语言
* Java虚拟机
* Java API类库

这三部分统称为JDK，JDK是用于支持Java程序开发的最小环境。

* Java API类库中的Java SE API子集
* Java虚拟机

这两部分统称为JRE。
